# JasonWu.github.io

Personal GitHub Pages site for Jason Wu. The site is a static collection of interactive pages organized into games, tools, simulators, and lab experiments, with a homepage that highlights live data widgets (moon phase, Bonn temperature, and Kp index).

## Sections

- **Home**: Landing page with live data widgets and category links to the rest of the site.
- **Games**: Interactive games like Sokoban, Doomish, and Family Dice.
- **Tools**: Utilities such as a translator, sand clock, and solar system demo.
- **Simulators**: Physics/system simulations like the FFT simulator.
- **Lab**: Experimental projects, starting with temperature conversion dials.

## Repository Layout

```
css/           Shared styles
index.html     Home page
games/         Game experiments and category page
tools/         Utility tools and category page
simulators/    Simulator demos and category page
lab/           Experimental projects and category page
```
